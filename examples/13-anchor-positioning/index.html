Anchor Positioning

CSS Anchor Positioning allows you to position elements relative to other elements (anchors) anywhere on the page. (Note: Requires browser support for CSS Anchor Positioning)

Usage

<!-- Define an anchor -->
<button class="anchor-name-[--my-button]">
  Click me
</button>

<!-- Position relative to anchor -->
<div class="position-anchor-[--my-button] position-area-bottom">
  I'm positioned below the button
</div>
